# Kotlin Compatibility

The Kotlin compiler plugin API is not a stable API, so not every version of Metro will work with every version of the Kotlin compiler.

| Kotlin version | Metro version |
|----------------|---------------|
| 2.2.10         | 0.4.0 -       |
| 2.2.0          | 0.4.0 -       |
| 2.1.21         | 0.3.1 - 0.3.8 |
| 2.1.20         | 0.1.2 - 0.3.0 |
